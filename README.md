# CineLog — Personal Movie Tracker

A full-stack movie tracking app where you can search for films, build a watchlist, rate and review movies, and filter your collection.

---

## Project Description

FilmLog is a personal movie tracker built with React (inspired by Letterboxd). Users can search a live movie catalog powered by the TMDB API, save films to their watchlist, mark them as watched, and leave ratings and reviews. Each user has their own private account so their data is saved and accessible anywhere.

---

## Planned Features

1. Movie Search — Search for any movie using the TMDB API with results showing poster, title, year, and genre.
2. Watchlist Management — Add movies to a personal watchlist and mark them as watched or unwatched.
3. Ratings and Reviews — Leave a star rating (1-5) and a short text review for any movie in your list.
4. Filter and Sort — Filter your watchlist by genre or watch status, or sort by rating or date added.

---

## Tech Stack

- Frontend: React (Vite)
- Auth and Database: Supabase (PostgreSQL + Auth)
- Movie Data: TMDB API
- Deployment: GitHub Pages
